Proyek

🤖 Hybrid Smart V3 — Trading Bot Algoritmik AI-Assisted

Peran: Strategist & Supervisor — Perancangan parameter strategi dan pengawasan operasional.

Bot trading kripto yang beroperasi 24 jam penuh di VPS dengan strategi hybrid multi-timeframe (4H/1D). Sistem menerapkan hard filter market meliputi ADX ≥ 20, EMA50 > EMA200, RSI dalam rentang 30-55, serta BTC correlation filter. Manajemen risiko mencakup trailing stop-loss dan take-profit bertingkat. Saya menentukan seluruh parameter strategi dan melakukan evaluasi keputusan trading secara berkala melalui Telegram, sementara AI mengeksekusi kode dan menjalankan monitoring.

🏨 Manajemen Infrastruktur Hotspot MikroTik AI-Assisted

Peran: Network Operator — Pengelolaan pengguna dan konfigurasi jaringan via AI.

Mengelola infrastruktur hotspot komersial dengan lebih dari 11 pengguna aktif yang terbagi dalam beberapa profil (kamar hunian, restoran, CCTV). Meliputi manajemen pengguna, pengaturan bandwidth, serta monitoring koneksi secara real-time. Seluruh perintah konfigurasi pada RouterOS 7 dieksekusi melalui AI agent yang terhubung via SSH.

🎨 Custom Captive Portal MikroTik AI-Assisted

Peran: Designer & Integrator — Perancangan dan implementasi halaman login hotspot yang disesuaikan dengan kebutuhan.

Mengembangkan halaman captive portal (hotspot login page) kustom untuk MikroTik RouterOS 7 — meliputi desain antarmuka, pengaturan redirect, serta integrasi dengan sistem hotspot yang sudah berjalan. Saya menentukan kebutuhan desain dan alur autentikasi, AI membantu menuliskan kode HTML, CSS, dan konfigurasi yang diperlukan.

📡 Sistem Monitoring Infrastruktur AI-Assisted

Peran: System Supervisor — Perancangan arsitektur monitoring dan evaluasi luaran sistem.

Sistem monitoring otomatis yang memeriksa status VPS, bot trading, dan konektivitas jaringan MikroTik secara berkala setiap 5 menit. Notifikasi dikirimkan secara real-time ke Telegram apabila terdeteksi anomali. Saya merancang kebutuhan sistem, AI mengimplementasikan script cron, health check, dan mekanisme notifikasi.

💻 Aplikasi Web Laravel AI-Assisted

Peran: Developer — Perancangan arsitektur aplikasi dan validasi luaran kode.

Pengembangan aplikasi web menggunakan framework Laravel — meliputi routing, database migration, Eloquent ORM, Blade templating, middleware, dan arsitektur MVC. Saya menentukan dependensi, merancang alur aplikasi, dan memvalidasi hasil. AI berperan dalam penulisan kode dan proses debugging.

⚙️ Implementasi Hermes Agent AI-Assisted

Peran: System Integrator — Instalasi dan konfigurasi AI agent pada infrastruktur VPS.

Instalasi dan konfigurasi Hermes Agent dari tahap awal — integrasi dengan provider DeepSeek dan 9router, penghubungan gateway Telegram, penjadwalan cron job untuk monitoring otomatis, serta pengembangan custom slash-command (/vps, /hotspot, /bot) yang memungkinkan eksekusi perintah secara interaktif melalui Telegram.

🔄 Implementasi 9Router — AI Gateway AI-Assisted

Peran: Infrastructure Administrator — Instalasi dan konfigurasi gateway multi-provider AI.

Setup 9router pada lingkungan Docker sebagai gateway yang menghubungkan lebih dari 40 penyedia AI dalam satu endpoint. Sistem berfungsi sebagai routing layer dengan mekanisme auto-fallback dan optimalisasi biaya. Hermes Agent terintegrasi dengan 9router sebagai custom provider untuk menjamin ketersediaan layanan AI secara berkelanjutan.

🖥️ Administrasi VPS Linux AI-Assisted

Peran: System Administrator — Pengelolaan server dan layanan berbasis cloud.

Pengelolaan VPS Ubuntu meliputi instalasi Docker, konfigurasi SSH key authentication, manajemen systemd service, firewall, monitoring resource (CPU, RAM, disk), penjadwalan cron job, dan manajemen logging. Seluruh operasi administratif dieksekusi melalui AI dengan supervisi langsung.

🌐 Konfigurasi Router MikroTik RB750Gr3 AI-Assisted

Peran: Network Administrator — Perancangan topologi dan konfigurasi jaringan.

Konfigurasi infrastruktur jaringan menggunakan MikroTik RB750Gr3 dengan RouterOS 7 — implementasi WireGuard VPN untuk akses remote aman dari VPS, hotspot server multi-profile, manajemen pengguna dan bandwidth, serta monitoring koneksi secara real-time. Seluruh konfigurasi dieksekusi melalui AI dengan pengawasan penuh terhadap aspek keamanan.
