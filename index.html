Proyek

🤖 Bot Trading Hybrid Smart V3

Peran: Strategi & Supervision — Saya menentukan parameter, mengevaluasi hasil, dan memutuskan kapan bot perlu diintervensi.

Bot trading kripto yang berjalan 24/7 di VPS. Saya memantau langsung lewat Telegram — cek status, baca log, evaluasi kenapa gak entry, dan adjust strategi. Bot pakai hybrid strategy dengan hard filter: ADX ≥ 20, EMA50 > EMA200, RSI 30-55, BTC correlation, trailing SL/TP. Saya yang putuskan parameternya.

🏨 Manajemen Hotspot MikroTik

Peran: Operator — Saya mengelola user hotspot, mengecek status, dan melakukan maintenance melalui AI agent.

Infrastruktur hotspot komersial dengan 11+ user aktif — multi-profile (kamar, resto, CCTV). Saya ngecek user aktif via command /hotspot, manage password, dan monitor koneksi. Semua operasional dilakukan lewat instruksi ke Hermes agent.

📡 Sistem Monitoring Real-time

Peran: Supervisor — Saya memastikan semua sistem berjalan dan mendapat notifikasi kalau ada masalah.

VPS, bot trading, dan jaringan MikroTik dimonitor otomatis dengan health check tiap 5 menit. Kalau ada yang mati, saya tau langsung dari Telegram. Saya juga bisa cek performa VPS kapan aja lewat command /vps.

⚙️ Setup & Operasional Hermes Agent

Peran: Operator — Saya menginstall, mengkonfigurasi, dan mengoperasikan AI agent di VPS.

Setup Hermes Agent dari awal — instalasi, konfigurasi provider (DeepSeek, OpenRouter, 9router), koneksi gateway Telegram, setup cron job untuk monitoring otomatis, dan pembuatan custom skill slash-command (/vps, /hotspot, /bot). Agent jalan 24/7 di Docker VPS dan bisa diakses kapan aja lewat Telegram.

🔄 Setup 9Router — AI API Router

Peran: Admin — Saya menginstall dan mengkonfigurasi 9router sebagai gateway AI provider.

Setup 9router di Docker — menghubungkan 40+ provider AI (gratis & berbayar) dalam satu endpoint. Hermes Agent terhubung ke 9router sebagai custom provider untuk routing otomatis, auto-fallback, dan hemat token. Jalan 24/7 sebagai service Docker terpisah, memastikan koneksi AI selalu available dengan biaya minimal.

🖥️ Setup & Maintenance VPS

Peran: Admin — Saya mengelola VPS Linux untuk hosting bot, agent, dan layanan lainnya.

Setup VPS Ubuntu dari nol — Docker container, SSH key authentication, systemd service management, firewall, monitoring resource (CPU/RAM/disk), cron job scheduling, dan logging. Manage semuanya via remote SSH dengan bantuan AI.

🌐 Setup & Manajemen Router MikroTik

Peran: Network Operator — Saya mengkonfigurasi dan memelihara router MikroTik RB750Gr3 (ROS 7).

Setup infrastruktur jaringan — WireGuard VPN untuk remote access dari VPS, hotspot server dengan multiple profile (kamar, resto, CCTV), manajemen user, bandwidth management, dan monitoring koneksi real-time. Akses remote aman via SSH dengan user read-only untuk keamanan.

🌐 Web Proxy MikroTik (Percobaan)

Peran: Network Operator — Saya mencoba setup web proxy di router untuk caching browsing.

Mencoba mengaktifkan web proxy MikroTik (port 8080) untuk caching konten web dan mempercepat browsing. Terkendala device-mode pada RB750Gr3 ROS 7 — proxy dinonaktifkan oleh firmware. Tetapi memahami konsep dan konfigurasi proxy di ekosistem MikroTik.
